# ADR 0004 — Capabilities

## Status
Accepted

## Context
Roles request Capabilities instead of tools. Maestro grants permissions and selects implementations. Deny-by-default prevents privilege escalation.

## Decision
Roles request Capabilities instead of tools. Maestro grants permissions and selects implementations. Deny-by-default prevents privilege escalation.

## Consequences
Improves modularity, portability and long-term maintainability.
