# Web UI

Version: 0.1

## Purpose

The Web UI visualizes the Maestro control plane.

The UI does not contain orchestration logic.

## Main Views

- Dashboard
- Projects
- Executions
- Workflow Graph
- Work Items
- Reviews
- Artifacts
- Knowledge
- Providers
- Agents
