# ADR 0007 — Provider Interface

## Status
Accepted

## Context
Providers abstract model runtimes behind stable contracts. Domain code never references Ollama, OpenAI or Codex directly.

## Decision
Providers abstract model runtimes behind stable contracts. Domain code never references Ollama, OpenAI or Codex directly.

## Consequences
Improves modularity, portability and long-term maintainability.
