# ADR 0001 — Control Plane

## Status
Accepted

## Context
Maestro is a control plane, not an AI assistant. It owns orchestration, scheduling, policies and persistence. Models only perform reasoning.

## Decision
Maestro is a control plane, not an AI assistant. It owns orchestration, scheduling, policies and persistence. Models only perform reasoning.

## Consequences
Improves modularity, portability and long-term maintainability.
