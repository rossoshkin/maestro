# ADR 0006 — Workspaces

## Status
Accepted

## Context
Every Execution runs inside an isolated Workspace. Source repositories remain untouched. Workspaces are disposable and reproducible.

## Decision
Every Execution runs inside an isolated Workspace. Source repositories remain untouched. Workspaces are disposable and reproducible.

## Consequences
Improves modularity, portability and long-term maintainability.
